## 🚀 Featured Projects

### 📅 College Alert — Android Event Management App
> Flutter · Firebase · Firestore

An Android app to manage and display college events with real-time data. Features Firebase Authentication, categorized event sections (Today, Upcoming, Important), in-app reminders, a profile screen, and a dark mode toggle.

---

### 🎨 ArtHub — Digital Art Gallery App
> Flutter · Firebase · Cloudinary

A mobile platform for uploading, exploring, and interacting with digital artworks. Integrated Firebase Auth, Firestore for content management, and Cloudinary for media storage. Artwork screens support likes, comments, and view counts.

---

> Flutter · Firebase · REST APIs · GetX

A feature-rich trip planning app with AI-assisted packing lists and survival guidance via REST APIs. Built with GetX state management, Firestore, Cloudinary, and reusable UI components following a clean, structured architecture.
